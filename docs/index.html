Apa itu SimpleLiteDB?

SimpleLiteDB adalah Database Management Suite berbasis web yang dirancang khusus untuk SQLite. Aplikasi ini bertindak sebagai jembatan (bridge) yang mengubah database SQLite lokal Anda menjadi layanan HTTP yang bisa diakses dari mana saja secara aman.

Bukan sekadar penampil data biasa, SimpleLiteDB menyediakan antarmuka modern (Dashboard), Terminal interaktif, dan Visualisasi Relasi (Schema Map) dalam satu paket ringan.

Kenapa Dibuat?

Banyak developer menghadapi masalah saat ingin menggunakan SQLite di lingkungan remote (VPS) atau ingin membagikan akses database tanpa memberikan akses SSH penuh.

SimpleLiteDB dibuat untuk:

  • Menghilangkan Kerumitan: Tidak perlu menginstal software client database yang berat.
  • Aksesibilitas: Mengelola database cukup melalui browser.
  • Modernisasi: Memberikan pengalaman manajemen database ala SaaS (seperti Supabase atau PlanetScale) namun untuk SQLite yang jauh lebih hemat sumber daya.

Untuk Apa?

Backend Sederhana

Sangat cocok untuk aplikasi web kecil, IoT, atau aplikasi mobile yang butuh database cloud tanpa ribet.

Shared Access

Berbagi akses database ke tim melalui API Key unik per database tanpa resiko keamanan sistem.

Cara Instalasi

# 1. Clone repository
git clone https://github.com/dnysaz/simple-lite-db

# 2. Inisialisasi Environment
./slite init

# 3. Jalankan Service
./slite start db

Menggunakan API

Setelah database dibuat, Anda bisa langsung memanggilnya dari aplikasi apapun menggunakan request HTTP POST sederhana.
