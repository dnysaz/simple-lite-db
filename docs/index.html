What is SimpleLiteDB?

SimpleLiteDB is a Database Management Suite specifically designed for SQLite. It acts as a secure bridge that turns your local SQLite files into a multi-tenant HTTP service accessible from anywhere.

With an integrated Admin Dashboard, Interactive Schema Map, and Web Terminal, it provides a "Cloud Database" experience while keeping the simplicity and cost-efficiency of SQLite.

Why use it?

Native SQLite is great, but it lacks remote access and multi-user capabilities. SimpleLiteDB solves this by providing:

  • Remote Management: Manage your databases via browser without SSH.
  • Multi-Tenancy: Create multiple isolated databases, each with its own API Key.
  • Zero Dependencies: No need for complex SQL drivers in your frontend or serverless functions.

Perfect for Serverless (Vercel, Netlify)

Modern platforms like Vercel, Netlify, or Cloudflare Workers don't allow local SQLite file writes because of their read-only file systems.

SimpleLiteDB enables you to use SQLite in these environments by hosting the database on a VPS and accessing it via standard HTTP requests. You get the speed of SQLite with the global reach of Serverless.

Quick Setup

# 1. Clone repository
git clone https://github.com/dnysaz/simple-lite-db

# 2. Initialize Environment
./slite init

# 3. Start Database Service
./slite start db

API Code Examples

Query your database from any application using standard fetch requests.

JavaScript (Fetch) 
const response = await fetch('http://YOUR_VPS_IP:5117/query', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_API_KEY'
    },
    body: JSON.stringify({
        database: 'my_db',
        sql: 'SELECT * FROM users WHERE active = ?',
        params: [true]
    })
});

const data = await response.json();
console.log(data.rows);
Python (requests) 
import requests

url = "http://YOUR_VPS_IP:5117/query"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}
payload = {
    "database": "my_db",
    "sql": "SELECT * FROM users"
}

res = requests.post(url, json=payload, headers=headers)
print(res.json()['rows'])
